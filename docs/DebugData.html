(static) fullDebugData

Determine if full details about the CallingClassInstance should be printed along with normal details.

This sets the value for the class (every DebugData object/instance). DebugData#fullDebugData only sets the value for one DebugData object/instance.


True means print full details of the CallingClassInstance as JSON.stringify(...)


False means do not print full details.
