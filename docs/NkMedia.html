Determines if peerConn can have more than one MediaStream at a given time. PeerConn allows this, BUT it causes problems in most cases. This is turned off (false) by default to prevent many problems. Only set this to true if you really know what you are doing.

  • true = Reset and get Ice Candidates every time an offer or answer is processed.
  • false = Only do Ice Candidates the first time.
